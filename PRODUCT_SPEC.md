# Memory Cards Product Spec

## One-liner
Memory Cards is an offline-first mobile photo memory game. It helps users revisit old photos, record memories, and lightly organize their camera roll without uploading or modifying original photos.

## Non-goals
- Not a cloud photo backup app.
- Not a social sharing app.
- Not a full photo cleaner.
- Not an AI autobiography generator.
- Not a real photo deletion tool in MVP.

## Privacy Principles
- Photos are read-only.
- Original photos are never modified.
- Audio is stored locally.
- Memory records are stored locally.
- No network calls in MVP.
- Export is user-initiated.

## MVP Loop
Open app → draw a photo card → answer/record/mark/skip → save local memory record → next card.

## Primary UX Tone
Entry feels like a casual game.
Stored output behaves like a serious tool.
No pressure, no forced streaks, no guilt.
